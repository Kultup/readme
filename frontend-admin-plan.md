# FRONTEND ADMIN (React)

**Призначення:** Веб адмін панель для управління контентом, користувачами, перегляду статистики та налаштування системи.

## Технологічний стек

- React (CRA або Next.js)
- React Router
- Material-UI / Ant Design
- Axios для API запитів
- Chart.js / Recharts для графіків
- React Query / SWR для кешування даних
- React Hook Form для форм
- react-dropzone або react-image-crop для завантаження та обробки зображень
